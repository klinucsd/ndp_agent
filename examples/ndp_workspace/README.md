# NDP Workspace Example

Example requests sent to DeepNDP for interacting with NDP JupyterHub workspaces via the `ndp-workspaces` skill. All files in this directory were generated by DeepNDP in response to the requests below.

---

## Request

```
Please list all data challenges via NDP workspace API.

Please show the details about the data challenge "Example Data Challenge and Onborading"

Please create a new folder under ~/work and download all data and code of the Sprint 1 of this
challenge into the new folder in a well organized way.

Could you please solve all the tasked in the notebook sprint1_1-4_team-name.ipynb in Module A?
```

## Generated Files

| File | Description |
|------|-------------|
| `sprint1_1-4_result.ipynb` | Notebook with solutions to all tasks in Module A of Sprint 1 |
