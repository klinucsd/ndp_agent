# NDP Nginx Log Analysis

This example demonstrates using **DeepNDP** to analyze National Data Platform (NDP) Nginx web logs and extract insights about user activity.

## Files

| File | Description |
|------|-------------|
| `ndp.log` | Nginx access log excerpt (3,447 entries) from NDP |
| `analyze_web_logs.ipynb` | Jupyter Notebook generated by DeepNDP to answer the analysis questions below |

## Original Task

```
The file ndp.log is extracted from our Nginx logs.

Please create a Jupyter Notebook to parse this log file and answer the following questions:

-- How many unique users accessed the system?
-- How many datasets were accessed, counted uniquely?
-- How many searches were made?

Please add a cell with "! pip install -q ..." and visualizations. Also, verify the notebook runs properly.
```
